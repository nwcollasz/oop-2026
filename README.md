# Exercícios de OOP

Repositório de exercícios de Programação Orientada a Objetos em C#.

## 📂 Pastas

- **arquivos-io:** Leitura de arquivos (stream reader, streamwriter, directoryinfo, fileblock) 
- **composicao:** Conceito de composição (herança, polimorfismo, try-catch)
- **expressoes:** Expressões e consultas (LINQ, lambda)
- **generics:** Uso de Generics e Dictionaries (gethashcode, conjuntos)
- **interfaces:** Interfaces (IComparable)
- **logica:** Exercícios de lógica (if-else, for, foreach, while)
